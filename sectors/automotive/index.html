solution

Miphawell System Layer(mSL) software library basically, provides device discoveryunder certain grouping category and a transparent device to device (P2P) directdata exchange (network permitting) facility for applications, without having toworry about underlying transport or prevailing network. The data exchangebetween two devices (regardless of their network residence), whenever possibleis carried out through direct communication between devices, rather than alwaysinvolving an explicit public node as data relay. Mimik's proprietary signallingand Virtual Smart Selective Session Transport Protocol(VS3TP) is used forexchanging data among nodes in mSL. Application-generated TCP data sessionsbetween two devices are always terminated locally (short-circuited) in proxyend-points at both the edge nodes and are translated to VS3TP for transmission.The resulting VS3TP payloads are then sent through a virtual transport channelto the destination devices. The same channel is multiplexed for all local TCPsessions with aggregated session management and congestion control. Sessionsare immune to network changes and outages and as a result, data transfercontinues to resume, across device network handover, without any further effortor coordination on the application layer.
